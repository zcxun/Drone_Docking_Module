# 採購與準備清單

## 第一批必買

| 類別 | 項目 | 用途 | 備註 |
| --- | --- | --- | --- |
| 伴隨電腦 | Raspberry Pi 5 或 Jetson Orin Nano | 視覺與狀態機 | Pi 5 較省成本；Jetson 較適合後續深度視覺 |
| 相機 | 下視 USB/CSI 相機 | AprilTag/ArUco 偵測 | 需固定焦距與可重複安裝角度 |
| 標記 | AprilTag/ArUco 標記板 | 目標定位 | 桌上治具先用高對比紙板或壓克力板 |
| 測距 | 下視 ToF/LiDAR | 近地高度與下降保護 | 必須實測有效距離與反光表面表現 |
| 電源 | 5V/6V 高品質 BEC | 伴隨電腦、伺服、感測供電 | 不可直接吃 Pixhawk 連接埠供電 |
| 線材 | TELEM/UART、JST-GH、杜邦線 | Pixhawk 與伴隨電腦/感測器連接 | 需要拉力固定與防鬆 |

## 機構第一批

| 項目 | 用途 | 驗收重點 |
| --- | --- | --- |
| 3D 列印導正漏斗/錐面 | 容忍水平偏移 | 不刮傷、不卡死、入口夠大 |
| 定位柱/導軌 | 重複定位 | yaw 偏差下仍可導入 |
| 彈簧/橡膠/TPU | 緩衝 | 接觸後無硬撞擊 |
| 金屬插銷/伺服鎖扣 | 鎖定 | 斷電狀態不可意外釋放 |
| 微動開關 | 接觸或 seated 判斷 | 必須有機械餘裕 |
| 霍爾感測器與磁鐵 | 第二鎖定訊號 | 不依賴單一開關 |
| 壓力感測片 | 選配 | 用於接觸力趨勢，不作唯一安全判定 |

## 控制與安全

- ESP32/Arduino：彙整感測訊號，後續可用 UART/I2C 回傳伴隨電腦。
- 伺服馬達或小型線性致動器：驅動鎖扣。
- 獨立保險絲、電源開關、急停線路：避免測試時短路或失控。
- 遙控器、接收機、遙測電台、蜂鳴器、安全開關：飛控基本安全設備。
- 槳保護、備用槳、LiPo 充電器、防火袋：低空測試基本裝備。

## 測試設備

- 電子秤：量全備重量、模組重量。
- 推力測試架或吊秤：確認推力餘裕。
- 萬用表與桌上電源供應器：量測 BEC、伺服電流、感測器輸出。
- 假太陽能板表面：先驗證反光與接觸，不用真板。
- 軟墊測試區、繫留線或安全籠：低空風險控制。

## 暫緩採購

- RTK、UWB、動作捕捉。
- 工業級 LiDAR。
- 電磁鐵主鎖。
- 真太陽能板場地與真清潔模組大改版。

上述項目需等輕載桌上與低空繫留 demo 穩定後再評估。
