# 風險與盲點清單

| 風險 | 影響 | 第一版對策 |
| --- | --- | --- |
| F450 載重不足 | 無法穩定 hover 或掛載後控制變差 | 先用 <=300 g 假負載，先量懸停油門與推力餘裕 |
| 下洗吹動模組或標記 | 桌上成功但飛行失敗 | 低空繫留時測標記與模組受風狀況 |
| 太陽能板反光 | 視覺辨識失敗、測距失真 | 第一版不用真板，只用假表面測反光 |
| 太陽能板脆弱 | 撞擊刮傷或破裂 | 第一版不接觸真太陽能板 |
| 鎖扣單點失效 | 誤以為已鎖定後起飛 | 必須雙感測確認，不通過單一開關 |
| 電磁鐵耗電與發熱 | 失電掉落、磁干擾 | 第一版不用電磁鐵作主鎖 |
| 伺服供電不足 | brownout 或 Pixhawk 重啟 | 伺服與伴隨電腦使用獨立 BEC |
| 測距器在近距離失真 | 下降控制錯誤 | 實測有效範圍，超出範圍即 abort |
| 目標遺失 | 無人機繼續下降到錯誤位置 | target timeout 後 abort/hover |
| 接觸時偏移過大 | 機構卡死或翻倒 | contact 前必須已在允許偏移內 |
| 鎖定過程卡住 | 燒伺服或撞機 | 鎖定與導正都有 timeout |
| 飛手無法即時接管 | 測試危險升高 | 遙控器 override 必須在每次測試前確認 |
| 感測線鬆脫 | 誤判狀態 | 線材固定、拉力釋放、故障注入測試 |
| 重心偏移 | 起飛不穩 | 每次改模組後量重心與 hover 行為 |
| 法規/場地問題 | 測試中止或危險 | 只在安全場地、低空、繫留、受控人員下進行 |

## Abort 原則

任何無法確認安全的狀態都要停止自動對接。第一版寧可多 abort，不追求連續成功率。
